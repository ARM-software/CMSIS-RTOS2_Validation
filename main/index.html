Introduction

This manual explains the scope and the usage of the software pack for CMSIS-RTOS2 Validation. CMSIS-RTOS2 API is a generic RTOS2 interface for ARM® Cortex®-M processor-based devices.

The software pack for CMSIS-RTOS2 validation provides:

  • Configurable validation tests

The CMSIS-RTOS2 validation tests and verifies:

  • Kernel Functions - Verify kernel information and control calls
  • Thread Functions - Create multiple threads, terminate, restart, yield, join, detach, change priorities
  • Thread Flags Functions - Verify thread flags
  • Generic Wait Functions - Call generic wait functions
  • Timer Functions - Create periodic and one-shot timers
  • Event Flags Functions - Verify event flags
  • Mutex Functions - Synchronize resource access
  • Semaphore Functions - Verify access shared resources
  • Memory Pool Functions - Verify memory pool allocation
  • Message Queue Functions - Exchange messages between threads

This manual contains the following chapters:

  • Reading Test Results - Explains how to interpret the test results.
  • Configuration - Contains information about test suite configuration options
  • Reference - Explains the individual CMSIS-RTOS2 API tests.

This manual assumes that you are familiar with Keil MDK. Refer to MDK Version 5 - Getting Started for additional information.
